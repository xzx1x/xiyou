# web-portal

Next.js + TypeScript 前端项目，面向学生、心理咨询师与管理员三类用户。建议使用 Bun 管理依赖与脚本。

## 目录占位
- `app/`：Next.js App Router 页面及路由
- `components/`：通用 UI 组件
- `features/`：按业务模块划分的视图与逻辑
- `lib/`：API 封装、工具函数、权限控制等
- `public/`：静态资源（图标、配置、国际化文件等）

## 推荐脚本（待初始化后添加）
- `bun dev`：本地开发
- `bun lint`：ESLint 检查
- `bun test`：前端测试
- `bun build`：生产构建
